[조작법]

1. 배틀넷 채널에서

*. 조작법

  F2, F3, F4 : wDetector의 제 1채널, 제 2채널, 제 3채널로 이동합니다. (Shift와 함께 누르면 1시간 이전 채널로 이동됩니다.)
  F5 : wDetector의 글에 연결된 홈페이지로 이동합니다.
  Shift + F5 : wDetector의 다른 글을 볼 수 있습니다.
  F6 : wDetector.ini의 MyChannel1에 설정된 채널로 이동합니다. (Shift와 함께 누르면 MyChannel2 채널로 이동됩니다.)
  F7 : 방을 따라갈 친구의 아이디를 설정합니다. (채팅창에 따라갈 친구 아이디를 먼저 입력해야 합니다.)
  F8 : 친구가 있는 방을 따라갑니다. (친구가 방에 들어갔을 때 누릅니다.)

  /filter : wDetectorFilter.txt 필터링 파일을 열거나 필터링할 단어를 추가할 수 있습니다.



2. 방에서 기다릴 때

*. 조작법

  F5 : 자신이 방장일 경우에 방을 새로고침합니다.
  Ctrl + 키패드 1~8 : 특정(1~8) 플레이어의 아이디를 복사합니다.
  Alt + 키패드 1~8 : 특정(1~8) 플레이어의 전적을 봅니다. (/stats)
  Win + 키패드 1~8 : 특정(1~8) 플레이어의 귓말을 수신거부합니다. (/ignore)
  키패드 '+' : 자동 방 새로고침을 켜고 끕니다.



3. 게임 플레이중에

*. 조작법

  키패드 '0' : 플레이어 리스트 (전적, 승률, APM, 종족, 핵 사용 목록) 등을 보여줍니다.
  키패드 '.' : 네트워크 리스트 (IP, Ping, 재전송 요청 빈도수, 패킷 정보) 등을 보여줍니다.
  키패드 '1~8' : 핵을 사용하는 특정(1~8) 플레이어를 다른 플레이어들에게 채팅창으로 알립니다.
  키패드 '9' : 핵을 사용하는 모든 플레이어들을 다른 플레이어들에게 채팅창으로 알립니다.
  Ctrl + 키패드 '0' : 플레이어 리스트를 리플레이에 기록합니다.
  Ctrl + 키패드 '1~8' : 특정(1~8) 플레이어의 아이디를 복사합니다.
  Alt + 키패드 '1~8' : 특정(1~8) 플레이어에게 화면을 보여달라고 요청합니다.
  Win + 키패드 '1~8' : 특정(1~8) 플레이어의 귓말을 수신거부합니다. (/ignore)
  Ctrl + Alt + 키패드 '1~8' : 특정(1~8) 플레이어에게 화면을 보여주거나 보여주고 있던 화면을 끕니다.
  키패드 '-' : 드랍창이 떴을 때 창을 보이거나 숨길 수 있습니다.
  키패드 '*' : Oblivion 핵 사용자를 추출해냅니다. (누른 후 잠시 동안 컨트롤을 해서는 안 됩니다.)
  키패드 '/' : 시계를 보이거나 숨길 수 있습니다.
  키패드 '+' : 채팅 로그를 열거나 닫을 수 있습니다.
  Ctrl + 키패드 '+' : 채팅 로그를 삭제합니다.

(wDetector.ini 파일에서 UseFuncKey 옵션을 1로 변경하면 F7, F8, F9, F11, F12 키로도 대신 사용할 수 있습니다.)

*. 추가 조작법

  Ctrl + P : 플레이어 리스트 보기 (키패드 '0')
  Ctrl + N : 네트워크 리스트 보기 (키패드 '.')
  Ctrl + I : 핵 사용자 알리기 (키패드 '9')
  Ctrl + L : 채팅 로그 보기 (키패드 '+')
  Ctrl + D : 드랍창 보이기, 숨기기 (키패드 '-')
  Ctrl + O : Oblivion 핵 사용자 추출 (키패드 '*')
  Ctrl + T : 게임중 시계 보이기, 숨기기 (키패드 '/')
